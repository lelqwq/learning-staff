打开 man 手册:
K 键：当光标位于你想要查询的函数名上时，在正常模式下按下大写的 K (Shift+k)
这个命令会尝试根据当前光标下的关键词运行一个外部手册页程序（通常是 man 命令）来查找相关信息


查找定义：
使用 gd 命令可以在当前文件中跳转到光标下的变量或函数的定义处。
如果你安装了 ctags 或者类似的标签管理工具（如 cscope），可以使用 Ctrl-] 跳转到光标下单词的定义位置，并用 Ctrl-t 返回。


自动补全：
在插入模式下，可以通过按下 Ctrl-n 或 Ctrl-p 来进行基于当前文档和其他打开文档上下文的代码补全。
使用插件如 YouCompleteMe 或者 coc.nvim 可以提供更强大的自动补全功能，包括语义补全。


快速移动：
使用 w, b 分别向前向后按单词移动；e 移动到单词末尾。
使用 { 和 } 可以快速跳过段落（对于注释和空行很有用）。
在文件内使用 gg 到达文件开头，G 到达文件末尾；数字前缀加上 G 可以直接跳到指定行号。


搜索与替换：
/pattern 搜索文本模式，n 和 N 分别用于下一个和上一个匹配项。
:%s/old/new/g 替换整个文件中的旧字符串为新字符串。


折叠代码：
zc 折叠光标所在的代码块，zo 展开折叠的代码块。
za 用于切换当前折叠状态（展开或收起）。


编译与运行：
使用 :make 编译项目（需要正确配置 makeprg 和 errorformat 等选项）。
运行 :copen 查看编译错误列表，并通过 :cn 和 :cp 浏览错误。


窗口操作：
Ctrl-w s 或 Ctrl-w v 分别水平或垂直分割窗口。
Ctrl-w w 在窗口间切换焦点，或者使用方向键组合 Ctrl-w 向指定方向移动。


标记：
使用 m[a-z] 设置本地标记，'[a-z] 跳转到标记的位置。
ma 设置标记 a，然后你可以用 'a 回到这个位置。


宏录制：
使用 q[a-z] 开始录制宏到寄存器 a-z，q 结束录制。使用 @[a-z] 回放录制的动作。
